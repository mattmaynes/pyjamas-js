Class: Version

Pyjamas.Version

new Version()

Collection of helper functions for handling version numbers. This class contains a set of static functions that deal with Major-Minor-Patch version identifiers.
Source:

Example
1.2.3

1 is the major version
2 is the minor version
3 is the patch version
