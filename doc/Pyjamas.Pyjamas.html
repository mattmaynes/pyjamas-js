(static) Pyjamas#upgrade(version, upgrader) → {Pyjamas.Pyjamas}

Registers an upgrade function to this Pyjamas instance. An upgrade function must accept a copy of a old version of a save and the current instance. It should either mutate the current instance of the save or return an upgraded version.
